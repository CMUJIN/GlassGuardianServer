Glass Guardian Use Tutorial

1) Go to Glass Home Screen


2) Start Guardian App(two ways)

a) Voice command - say "ok glass", pop up command list, then say "Protect me" start the app


b) Tab to start - tab on the on screen, scroll all the command until seeing "Protect me " command, tap again.


3) after app started, user see "Protect mode on" on screen, every 30 seconds, it records the environment data automatically on the backend


4) trigger alert(two ways)

a) manually trigger it by tapping on the "Protect me" screen, it pops up menu "Send Alert!"


b) shake your head quick and sharply, the alert will be triggered without hands


5) once alert triggered, user has 10s to cancel it or send


6) if user want to send alert, user can either do nothing until countdown to 0 or swipe forward on the touch bar to send it immediately


7) if user want to cancel the alert, manually tap on the touch bar and tap again to cancel


8) if user want to exit the app, tap on "protect mode on" screen and swipe to "Stop Guardian" menu
