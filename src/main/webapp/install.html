How to install Glass Guardian

1. Download Glass Guardian Apk file from download link.

2. On Google Glass, navigate to Settings / Device Info and turn on debug.

3. Connect Google Glass to a computer, and make sure ADB is set up.

4. Open a terminal on the folder that contains GlassGuardian.apk.

5. Run the command: adb install GlassGuardian.apk.

6. Say "Ok Glass, protect me" to open the app.
